介绍

peewee

Peewee是一个简单而小巧的orm。它具有很少（但富有表现力）的概念，使其易于学习和直观使用。

  • 一个简单而富有表现力的ORM
  • 适用于python2.7和3.4+（使用3.6开发）
  • 支持sqlite, mysql 和 postgresql
  • 拥有大量的扩展
