# 安装和测试<div id="introduce"></div>

大多数用户只想安装PyPi上托管的最新版本：

```python
pip install peewee

```



# 使用git安装<div id="git"></div>

该项目托管在https://github.com/coleifer/peewee，可以使用git安装：

```python
git clone https://

```


# 
运行测试套件

通过运行测试套件来测试已安装的。

```python
python runtests.py

```
